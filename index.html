Conversão de Temperaturas: Kelvin, Fahrenheit e Celsius

Para converter temperaturas entre Kelvin (K), Fahrenheit (F) e Celsius (C), utilizamos fórmulas específicas que relacionam essas três escalas. É importante lembrar que cada escala possui um ponto de referência diferente, como o ponto de congelamento e ebulição da água.

Fórmulas de Conversão

Aqui estão as principais fórmulas para converter entre as escalas:
Celsius (C) para Fahrenheit (F): F=(C×1.8)+32
Para converter Celsius para Fahrenheit, multiplicamos a temperatura em Celsius por 1.8 e adicionamos 32.
Fahrenheit (F) para Celsius (C): C=(F−32)/1.8
Para converter Fahrenheit para Celsius, subtraímos 32 da temperatura em Fahrenheit e dividimos o resultado por 1.8.
Celsius (C) para Kelvin (K): K=C+273.15
Para converter Celsius para Kelvin, simplesmente adicionamos 273.15 à temperatura em Celsius. Note que 0 Kelvin (zero absoluto) é o ponto onde não há mais energia térmica.
Kelvin (K) para Celsius (C): C=K−273.15
Para converter Kelvin para Celsius, subtraímos 273.15 da temperatura em Kelvin.
Fahrenheit (F) para Kelvin (K): K=(F−32)/1.8+273.15
Para converter Fahrenheit para Kelvin, primeiro convertemos Fahrenheit para Celsius e depois convertemos o resultado para Kelvin.
Kelvin (K) para Fahrenheit (F): F=(K−273.15)×1.8+32
Para converter Kelvin para Fahrenheit, primeiro convertemos Kelvin para Celsius e depois convertemos o resultado para Fahrenheit.

Exemplos

Vamos ver alguns exemplos para entender melhor:
Converter 25°C para Fahrenheit:
F=(25×1.8)+32
F=45+32
F=77°F
Converter 298.15 K para Celsius:
C=298.15−273.15
C=25°C
Converter 68°F para Celsius:
C=(68−32)/1.8
C=36/1.8
C=20°C
